# Map My World

## Description
This project is about create reviews of places and categories of places. The user can create, read, update and delete reviews of places and categories of places. The user can also see the reviews of places and categories of places.

## Prerequisites
- Docker installed on your machine.
- Git installed on your machine.
- A code editor installed on your machine.
- A web browser installed on your machine.

## Getting Started

### Clone the Repository
First, clone the repository to your local machine using the following command:
    
    git clone https://github.com/andyvzcode/Map-My-World
    

### Run the Application
To run the application, navigate to the project directory and run the following command:

    docker-compose up

### Access the Application
To access the application, open a web browser and navigate to the following URL:

    http://localhost:8080
    

### Documentation

To access the documentation, open a web browser and navigate to the following URL:

    http://localhost:8080/docs
